Lerne mit Spaß Programmieren!

Wofür ist diese Webseite gut:

Um mit spass programmieren zu lernen und seine Programmiermuskeln wachsen zu lassen muss man mit den Programmier beispielen rumspielen und genau das mache ich auf dieser Webseite ich nehme mir einfach beispiele und zeige euch wie man damit spielen kann und stelle kleine aufgaben dazu.

Was brauchst du um auf dieser Seite spass zu haben:

Eigentlich nur ein Computer und spass am spielen.

Am besten du installierst dir visual Studio Code hier ein Link für den download. Und hier noch ein Link für eine kurze einleitung im Youtube wie mans instaliert. Für eine ausführliche einleitung hast du hier den link hier link hier. Aber eigentlich ist das sehr einfach.
Theoretisch kannst du jeden text editor nehmen der dir gefählt nur dann wird es anders als auf den Bildschirm auschnitten aussehen. Die ich hier auf der Webseite zeige.

Und wir legen Los

Die Ersten Beispiele in Javascript findest du gleich weiter unten. HTML und CSS bekommst du auch noch nebenbei mit brauchst also keine vorkenntnisse.

Wir testen ob alles richtig instaliert worden ist. Wenn du visual Studio Code instaliert hast und es Startest sollte es so aussehen.

Jetzt zum CodeBeispiel 

          
<!DOCTYPE html>
<html>
<body>

<p id="vorstellung">Was kann man mit JavaScript so anstellen</p>

<button onclick="myFunction()">Click me</button>


<script>
function myFunction() {
  document.getElementById("vorstellung").innerHTML = "JavaScript kann HTML verändern!";
}
</script>

</body>
</html>

Dises beispiel kopierst du einfach in eine neue Datei im visual Studio Code speicherst diese am besten unter dem Namen jsbeispiel1.html dan müsste es so aussehen

Jetzt drückst du Strg + F1 . Dann müsste es in deinem Browser so Aussehen

Wenn du jetzt denn Click me Button drückst sollte es drauf reagieren etwa so


Glückwunsch

Wenn das alles klappt bist du bereit für die ersten Spielereien


Spielereien wie

-1 den ersten Text verändern
-2 die Schrift im Button ändern
-3 den Antwort Text verändern
-4 den Text aus dem nichts auftauchen lassen
-5 den Text unter dem Button auftauchen lassen
-6 Zwei Buttons machen

Man kann auch was ganz verrücktes anstellen wie 10 Buttons machen. Einen Ganzen Roman auftauchen lassen wenn man den Button drückt. Es ist alles möglich der Fantasie sind keine Grenzen gesetzt. Und der Verrücktheit auch nicht.
